# embedded-power-rail-5v-design
Engineered a low-noise, high-efficiency 5V power supply PCB with optimized component selection, ripple minimization, and robust layout design

⚡ High-Efficiency 5V Power Supply PCB

A compact, low-noise and high-efficiency 5V regulated power supply designed for embedded and analog applications.

This project goes beyond a basic regulator by focusing on **efficiency optimization, ripple reduction, and thermal stability**.

---

## 🔍 Overview

This PCB delivers a **stable 5V output** suitable for:
- Microcontrollers
- Embedded systems
- Sensor modules
- Analog circuits

Key focus was improving:
- Efficiency 🔋
- Voltage stability 📉
- Noise reduction 🔇

---

## ⚙️ Key Features

- ✅ Stable 5V output with minimal ripple
- ✅ Optimized efficiency (improved over standard linear designs)
- ✅ Proper decoupling and filtering
- ✅ Thermal-aware PCB layout
- ✅ Compact and robust design

---

## 🧠 Design Highlights

- Carefully selected regulator topology
- Optimized capacitor placement for noise suppression
- Ground plane design to reduce EMI
- Trace width calculated for current handling
- Efficiency improvements using better component selection

---

## 📊 Performance

| Parameter        | Value            |
|-----------------|------------------|
| Output Voltage  | 5V ± ~2%         |
| Efficiency      | ~97% |
| Ripple          | Low (10 mV – 50 mV)     |
| Input Range     | 6–25 V          |
